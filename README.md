# ATLAS

Geopolitical event timeline visualizer. Aggregates Wikipedia current events and historical fallback data, renders interactive horizontal timeline by region and event type, and produces strategic briefs via MiMo reasoning.

- Vanilla HTML/CSS/JS, no backend, no build step
- 32 hardcoded historical events (2020-2025) + live Wikipedia feed
- SVG timeline with region tracks and severity-scaled nodes
- MiMo reasoning slot for actor analysis, consequence forecasting, historical analogs
- MIT licensed

## Disclaimer

ATLAS aggregates from public sources. AI strategic briefs are heuristic interpretations and may oversimplify complex geopolitical dynamics. Always cross-reference with primary sources.
